FitControl é um sistema web de gerenciamento para academias, desenvolvido com foco em praticidade, organização e análise de dados.

O sistema permite:
- Cadastro de alunos
- Gerenciamento de planos e mensalidades
- Controle de presença
- Agenda de aulas
- Relatórios com dados e indicadores
- Dashboard com gráficos interativos

A interface foi projetada para ser simples, moderna e responsiva, facilitando o uso por gestores e funcionários de academias.

Tecnologias utilizadas:
- HTML
- CSS
- JavaScript
- Chart.js
